Ace – Student Management System

Tech Stack: C++, Object-Oriented Programming

Key Features:

  • Personal information management
  • Daily to-do list and schedule planning
  • Class and course tracking
  • Includes a small game for engagement

Reuse and Earn – Student Marketplace

Description: Reuse and Earn is a student marketplace designed for Banasthali Vidyapith where students can buy, sell, and exchange second-hand items. The platform reduces dependency on scattered WhatsApp groups and promotes sustainability.

Tech Stack: HTML, CSS, JavaScript

Key Features:

  • Centralized listing platform
  • Buy and sell second-hand items
  • Easy search functionality
  • Promotes sustainable student community

Tech Stack: Kotlin, Android Studio, Gradle

Key Features:

Key Features:

  • Smart grievance tracking
  • Automated categorization system
